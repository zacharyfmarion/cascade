# Code Of Conduct

Cascade is intended to be a welcoming, respectful project.

## Expected Behavior

Participants in this project are expected to:

- be respectful in language and tone
- assume good intent while still giving honest technical feedback
- focus criticism on ideas, code, and behavior rather than people
- help keep discussions constructive, especially when disagreeing
- make room for contributors with different levels of experience

## Unacceptable Behavior

The following behavior is not acceptable in project spaces:

- harassment, threats, or intimidation
- discriminatory or demeaning language
- personal attacks or repeated bad-faith engagement
- publishing private information without permission
- behavior that makes collaboration feel unsafe or hostile

## Enforcement

Project maintainers may remove comments, close discussions, reject contributions, or block participation when necessary to protect the community.

## Reporting

If something happens that should be reviewed by the maintainers, please report it through the project's GitHub channels with as much relevant context as you can provide.
Reports will be handled as respectfully and discreetly as possible.
