Cascade

Node-based image editing for the browser.

Cascade is a web-first image editor for building compositing and image-processing workflows as typed node graphs. It combines a Rust core, WebAssembly runtime, GPU-backed execution paths, and a desktop shell that shares the same engine.

Built for graph-based workflows

Assemble still-image pipelines visually, connect typed nodes, and iterate on non-linear image-processing graphs.

Rust engine, browser runtime

The core evaluator is written in Rust and runs in the browser through WebAssembly, with desktop support on the roadmap.

GPU and scripting support

Use GPU-backed execution paths where available, plus scripted GPU kernels for advanced processing workflows.

Core capabilities

  • Typed node graphs for image processing and compositing
  • Web-first editing with optional desktop builds
  • Still-image workflows, playback-oriented nodes, and AI-assisted tooling
  • Open-source codebase powered by Rust, React, and WebAssembly
